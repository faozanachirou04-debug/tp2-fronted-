# TP 2 Développement

## Projet : Gabon Digital Solutions (GDS)

### Présentation
Gabon Digital Solutions (GDS) est un site web d'entreprise fictive réalisé dans le cadre du TP 2 de développement web.

Le site contient :
- Une page d'accueil
- Une page À propos avec une équipe
- Une page Services
- Une page Contact avec un formulaire HTML5

### Technologies utilisées
- HTML5
- CSS3
